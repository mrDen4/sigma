Преимущества договора дарения:

1. Отсутствие обязанности по уплате налога на доходы физических лиц.

2. Сокращенный срок ожидания для последующего отчуждения по договору купли-продажи.

3. При заключении договора дарения доли недвижимости не требуется соблюдать правило о преимущественном праве покупки другими сособственниками.

4. Договор дарения может быть расторгнут дарителем в определенном законом случае.

5. На недвижимость, переданную по договору дарения в период брака, не распространяется режим общей совместной собственности супругов.

Основные недостатки договора дарения:

1. При заключении договора дарения между третьими лицами необходимо оплатить НДФЛ.

2. Договор дарения легче оспорить в суде и признать недействительным.

3. Запрещено дарение в отношениях между коммерческими организациями.

4. Дарение имущества, находящегося в общей совместной собственности, допускается только после получения согласия всех сособственников.

5. Возможность отмены договора дарения.
